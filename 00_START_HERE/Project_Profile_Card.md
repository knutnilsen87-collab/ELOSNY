## One-sentence description
Evida helps legal professionals turn document-heavy matters into source-bound facts, verified drafts, human-reviewed approvals and audit-ready export packages.

## Core problem
Legal teams spend too much time moving between documents, notes, drafts, source checks and review gates, while AI tools often hide uncertainty and make it hard to prove what an output is based on.

## Core user
Lawyers and legal reviewers handling document-heavy matters where source control, review, verification and auditability matter.

## Core job to be done
When a lawyer opens a matter, they need to understand what is known, what is missing, what is risky, what sources support the work, and what the next safe action is.

## MVP definition
The first complete product slice lets a user create a matter, import documents, extract source-bound facts and evidence, create a sourced draft, run verification, complete human review, and export a package containing the document, sources, verification result, review decision and audit summary.
